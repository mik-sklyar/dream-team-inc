# Dream-Team inc.

Групповой проект для курса Aston: Java с нуля.

## Описание:

Консольное приложение для найма и обработки списка сотрудников амбициозного стартапа по захвату мира.  Есть 3 способа получить данные:
- ввести вручную
- загрузить из файла
- использовать random генератор

Полученный список можно сортировать по разным критериям. Найти группы пользователей с одинаковым паролем.

## Технические детали:

Проект написал 4 бойцами за пару недель. Использовали максимально гибкую систему взаимодействия, чтобы не упороться и может даже местами кайфануть. В проекте использованы паттерны проектирования, интересные архитектурные решения и здравый смысл. Тем не менее **Language level - JDK-11**, чтобы будущие frontend/backend разработчики не привыкали к хорошему, чем пользоваться не смогут.

Помимо базового функционала реализованы:
- механизм quick sort с дополнительными условиями и фильтрацией
- собственная коллекция linked list, которая заменяет стандартный List<> без лишних приседаний
- многопоточный поиск по коллекции

Описание архитектуры проекта в презентации в [папке docs](https://github.com/mik-sklyar/dream-team-inc/tree/main/docs).
